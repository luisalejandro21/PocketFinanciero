# Alcance Versión 0.1 — PocketFinanciero

## Objetivo

Construir la base técnica inicial de PocketFinanciero antes del MVP.

Esta versión no está pensada para usuarios finales. Su objetivo es validar que la arquitectura, la estructura del proyecto y la comunicación entre frontend, backend y base de datos funcionen correctamente.

## Funcionalidades incluidas

- Configuración inicial del repositorio.
- Backend iniciado con Django.
- API REST iniciada con Django REST Framework.
- Base de datos PostgreSQL conectada.
- Frontend iniciado con React.
- Autenticación inicial con JWT.
- CRUD básico de:
  - cuentas
  - categorías
  - movimientos
- Documentación inicial del proyecto.

## Funcionalidades excluidas

- Recomendaciones inteligentes.
- Simulador de compras.
- Reportes avanzados.
- Importación bancaria.
- Inteligencia artificial.
- Aplicación móvil.
- Finanzas familiares.
- Integración con bancos.

## Criterio de término

La versión 0.1 estará terminada cuando:

- El usuario pueda registrarse e iniciar sesión.
- El backend pueda proteger rutas privadas.
- El frontend pueda conectarse al backend.
- Se puedan crear, listar, editar y eliminar cuentas, categorías y movimientos.
- La documentación inicial esté actualizada.
